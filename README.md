## Carbonara test

1. Installation

This project has been developed using Node 14

    ```shell
    npm i
    ```

2. Local development

   ```shell
   npm start
   ```

   The website will run on http://localhost:8000

3. UI

   The UI has been made using [Material UI v4](https://material-ui.com/)

4. Client Side

   The Client Side has been developed using
   [Gatsbyjs v3](https://www.gatsbyjs.com/)

5. Server side

   The only api `register` has been mocked using
   [Mock Service Workers](https://mswjs.io/)

6. Tests

   All the tests can be found in `./src/__tests__`, I've used jest as test
   engine and [Testing Library](https://testing-library.com/) as rendering
   engine

7. The website is hosted at
   [Github Pages](https://akallabet.github.io/carbonara/)
